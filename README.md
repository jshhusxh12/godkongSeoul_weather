# 서울의 날씨를 기록하는 파이썬 코드입니다.

```python
----- setup ----
1. https://openweathermap.org/ 해당 사이트에 접속한후 API KEY를 발급받은후, API 키를 입력해주세요.
2. pip install requests 모듈을 다운로드 해주세요. ( 해당 모듈이 없으면 작동이 안됩니다. )
```

```diff
==== Values ====
API_KEY : API 키 입력하는 변수
LAT : 서울의 위도
LON : 서울의 경도
URL : API URL 구성 변수
temp : 온도
feels_like : 체감 온도
weather : 날씨
description : 날씨의 상태
```
